Doodle Monster

Doodle Monster is a personal project I'm developing. It's a collaborative drawing game where players take turns adding lines to a drawing to create a wacky monster. Inspired by doodling on the back of placemats at diners.

The iOS app is written in Swift. The back end api is a NodeJS app hosted on Heroku.
